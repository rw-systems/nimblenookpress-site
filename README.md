# Nimble Nook Press — Marketing Site

Landing page for [nimblenookpress.com](https://www.nimblenookpress.com).

Built with Vite + React + TypeScript. Hosted on Vercel.

## Related

- **StoryForge app** → `app.nimblenookpress.com` (separate repo: `StoryForge/`)

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` auto-deploy via Vercel. The `vercel.json` at the root configures the build.

## Domain Setup (Vercel)

1. Add this repo as a Vercel project
2. Assign `www.nimblenookpress.com` (and `nimblenookpress.com`) as custom domains
3. StoryForge gets its own Vercel project → assign `app.nimblenookpress.com`
